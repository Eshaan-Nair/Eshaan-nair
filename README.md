## Featured Project

<div align="center">

### [ArcRift](https://github.com/Eshaan-Nair/ArcRift) — *The Memory Layer AI Was Missing*

[![Stars](https://img.shields.io/github/stars/Eshaan-Nair/ArcRift?style=social)](https://github.com/Eshaan-Nair/ArcRift/stargazers)
[![Forks](https://img.shields.io/github/forks/Eshaan-Nair/ArcRift?style=social)](https://github.com/Eshaan-Nair/ArcRift/network/members)
[![npm version](https://img.shields.io/npm/v/arcrift-setup?color=CB3837&label=npx%20arcrift-setup&logo=npm)](https://www.npmjs.com/package/arcrift-setup)
[![TypeScript](https://img.shields.io/badge/TypeScript-100%25-3178C6?logo=typescript&logoColor=white)](https://github.com/Eshaan-Nair/ArcRift)

</div>

> A **100% local, zero-cloud** persistent memory system that bridges your AI tools — browser chats (ChatGPT, Claude, Gemini) and IDE agents (Cursor, Windsurf) — into a single shared knowledge graph stored in SQLite.

| Feature | Details |
|---|---|
| **One-command setup** | `npx arcrift-setup` — zero config |
| **Hybrid RAG Engine** | Dense vectors + BM25 FTS5 keyword + HyDE — **90% recall@1 across 1,000 chunks** |
| **Surgical Trimming** | Intersection-based algorithm achieves **95% context compression** |
| **Dual Interface** | Chrome MV3 extension (SPA-aware auto-inject) + MCP server (7 tools) |
| **D3 Knowledge Graph** | Real-time visual dashboard of your memory nodes |
| **100% Private** | Everything runs locally via Ollama — zero data leaves your machine |
